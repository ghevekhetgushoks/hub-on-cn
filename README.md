# hub-on-cn

一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为若干静态 HTML 文件的集中存放与展示空间。

## 项目简介

本项目旨在提供一个简洁、无依赖的 HTML 页面归档场所。您可以直接通过 GitHub Pages 或其他静态托管方式访问仓库中存放的各个页面。每个页面均为独立的 HTML 文件，不依赖外部资源或后台服务。

## 目录说明

```
.
├── index.html          # 仓库首页（可选）
├── pages/              # 存放归档页面的目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── README.md           # 本文件
```

- **根目录**：通常放置一个简单的索引页（如 `index.html`），用于列出所有归档页面。
- **pages/**：所有独立 HTML 页面存放于此。文件名应具有描述性，便于识别内容。

## 页面归档说明

- 每个页面均为纯 HTML 文件，可包含 CSS、JavaScript 等内联资源。
- 页面内容不涉及特定网站或域名，仅作为通用信息展示或工具页面。
- 归档页面不包含任何跟踪、广告或外部请求（除非明确说明）。

## 维护说明

- 本仓库由维护者不定期更新，添加新的归档页面或修正现有页面。
- 欢迎提交 Issue 或 Pull Request 提出改进建议，但请注意不要添加与具体网站或域名相关的内容。
- 所有页面应遵守开源社区规范，不包含违法、侵权或恶意代码。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开 `pages/` 下的任意 HTML 文件，即可查看对应页面。
3. 若启用 GitHub Pages，可通过仓库设置中的 Pages 功能访问在线版本。
